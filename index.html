Color
Set background color using A/B/X/Y buttons, change it using the left stick
Paint
Set the brush color using A/B/X/Y buttons, move the brush using the left stick, change the brush size using the right stick
Stamp
Set the stamp color using A/B/X/Y buttons, move the stamp using the left stick, change the stamps size using the right stick, rotate the stamp using the L/R triggers, change the stamp shape using the d-pad and place a stamp using the L/R bumpers
